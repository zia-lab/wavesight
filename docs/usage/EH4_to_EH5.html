EH4_to_EH5¶

EH4_to_EH5.h5_handler(filename, mode)[source]¶
EH4_to_EH5.wave_to_vol(waveguide_id, zPropindex)[source]¶

This function takes the id for a given simulation and takes the fields that have been propagated across the metalens and propagate them along the z-axis by the provided distance.

The distance is provided as an index in the the zProps array which is determined from the start of the simulation.

The result is save into a new folder called EH5 in h5 format.

Parameters¶

waveguide_idstr

The label for the job.

zPropindex : int

None
