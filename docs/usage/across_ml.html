across_ml¶

This script is used to set up and launch FDTD simulations of the designed metalenses.

across_ml.meta_duct(waveguide_id, mode_idx)[source]¶

This function will take the id for a waveguide and the index for one of its propagating modes, and it will take the fields that have been propagated to the input plane of the metalens and will produce the fields that are transported across it.

across_ml.metasurf_designer(metalens_design_dict, post_z_center)[source]¶

Given a dictionary with the design parameters for a metalens, this function will return a list of MEEP objects that represent the metalens structure.

Parameters¶

metalens_design_dictdict

A dictionary with the design parameters for the metalens.

post_z_centerfloat

The z coordinate of the center of the metalens pillars.

Returns¶

post_forestlist

A list of MEEP objects that represent the cylindrical pillars that make up the metalens.
