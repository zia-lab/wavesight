fungenerators¶

This is a module that contains functions useful to calculate the propagating modes of a step index fiber.

fungenerators.hefungen(λfree, m, n1, n2, a)[source]¶

This function returns the eigenvalue function for HE(n,m) modes in a step index waveguide.

Parameters¶

λfreefloat

Free space wavelength in μm.

n1float

Core refractive index.

n2float

Cladding refractive index.

afloat

Core radius in μm.

mint

Order of the HE mode.

Returns¶

he : function

fungenerators.tefungen(λfree, n1, n2, a)[source]¶

This function returns the eigenvalue function for TE modes.
